vibe-stealer

Extract design tokens from any website. Colors, fonts, spacing, shadows.

v1.0.0

Install

npx vibe-stealer https://stripe.com

Color Palettes

Primary, neutral, accent colors extracted from computed styles

Typography

Font families, sizes, weights, line heights

Spacing Scale

Padding, margin, gap values normalized to a scale

Visual Effects

Shadows, borders, border-radius, blur

Output Formats

CSS Variables Tailwind Config JSON

Usage

# CSS variables (default)
node index.js https://stripe.com

# Tailwind config
node index.js https://stripe.com --format tailwind

# Save to file
node index.js https://linear.app -o linear.css

# Visible browser (debug)
node index.js https://vercel.com --headless false
